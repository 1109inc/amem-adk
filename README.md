# AMem-ADK

A research project mentor agent with a custom long-term memory backend inspired by A-Mem.

Goal:
Build an ADK agent that can remember, connect, and evolve project knowledge across sessions.

Phase 1:
- Create a basic Google ADK agent
- Run a simple conversation
- No custom memory yet

Later phases:
- Implement custom BaseMemoryService
- Add structured memory notes
- Add semantic links
- Add memory evolution
- Add graph-expanded retrieval
